# freee CLI Implementation Plan

> For agentic workers: REQUIRED SUB-SKILL: Use superpowers:subagent-driven-development (recommended) or superpowers:executing-plans to implement this plan task-by-task. Steps use checkbox (`- [ ]`) syntax for tracking.

Goal: Build a `freee` CLI that replaces MCP tools with ntn-style shell commands for freee accounting API, reusing existing auth/config/schema code.

Architecture: Add a new CLI entry point (`src/cli-app/`) alongside the existing MCP server. The CLI parses ntn-style input syntax (`key==value` for query, `key=value` for body, `-d` for JSON, `-X` for method override) and delegates to the existing `makeApiRequest()`. A new docs generator reads the full OpenAPI schema to produce references-style Markdown. The `freee` binary is registered in `package.json` `bin` alongside the existing `freee-mcp`.

Tech Stack: TypeScript, Bun (build + runtime), existing auth/config/openapi modules, vitest for testing.

```typescript
// src/cli-app/router.ts
export interface ParsedCommand {
  group: string;
  command: string;
  args: string[];
}

const API_SERVICES = ['accounting', 'hr', 'invoice', 'pm', 'sm'];

export function parseCommand(argv: string[]): ParsedCommand {
  if (argv.length === 0) {
    return { group: 'help', command: 'help', args: [] };
  }

  const [group, ...rest] = argv;

  // API service subcommands: if second arg starts with '/' it's an API path
  if (API_SERVICES.includes(group)) {
    if (rest.length === 0 || rest[0] === 'ls') {
      return { group, command: 'ls', args: rest.slice(rest[0] === 'ls' ? 1 : 0) };
    }
    if (rest[0].startsWith('/')) {
      return { group, command: 'api', args: rest };
    }
    return { group, command: rest[0], args: rest.slice(1) };
  }

  // Other groups: auth, company
  // If no subcommand given (e.g., bare "freee auth"), command equals group
  const command = rest[0] ?? group;
  const args = rest.length > 0 ? rest.slice(1) : [];

  return { group, command, args };
}
```

// src/cli-app/input-parser.ts
export interface ApiInput {
  path: string;
  method: string | undefined;
  query: Record<string, string>;
  body: Record<string, unknown> | undefined;
  flags: string[];
}

export function parseApiInput(args: string[]): ApiInput {
  const path = args[0];
  const query: Record<string, string> = {};
  let body: Record<string, unknown> | undefined;
  let method: string | undefined;
  const flags: string[] = [];

  let i = 1;
  while (i < args.length) {
    const arg = args[i];

    // -X METHOD
    if (arg === '-X' && i + 1 < args.length) {
      method = args[i + 1].toUpperCase();
      i += 2;
      continue;
    }

    // -d JSON
    if (arg === '-d' && i + 1 < args.length) {
      body = JSON.parse(args[i + 1]);
      i += 2;
      continue;
    }

    // Flags: --help, --docs, --spec
    if (arg.startsWith('--')) {
      flags.push(arg);
      i++;
      continue;
    }

    // key==value (query param)
    const queryMatch = arg.match(/^([^=]+)==(.*)$/);
    if (queryMatch) {
      query[queryMatch[1]] = queryMatch[2];
      i++;
      continue;
    }

    // key:=json (typed body field)
    const typedMatch = arg.match(/^([^:]+):=(.*)$/);
    if (typedMatch) {
      if (!body) body = {};
      body[typedMatch[1]] = JSON.parse(typedMatch[2]);
      i++;
      continue;
    }

    // key[nested]=value or key=value (body field)
    const bodyMatch = arg.match(/^([^=]+)=(.*)$/);
    if (bodyMatch) {
      if (!body) body = {};
      setNestedValue(body, bodyMatch[1], bodyMatch[2]);
      i++;
      continue;
    }

    i++;
  }

  return { path, method, query, body, flags };
}

function setNestedValue(obj: Record<string, unknown>, key: string, value: string): void {
  // Handle bracket notation: details[0][amount] -> nested assignment
  const parts = key.replace(/\]/g, '').split('[');

  if (parts.length === 1) {
    obj[key] = value;
    return;
  }

  let current: Record<string, unknown> = obj;
  for (let i = 0; i < parts.length - 1; i++) {
    const part = parts[i];
    const nextPart = parts[i + 1];
    const isNextIndex = /^\d+$/.test(nextPart);

    if (!(part in current)) {
      current[part] = isNextIndex ? [] : {};
    }

    const next = current[part];
    if (Array.isArray(next)) {
      const idx = Number.parseInt(nextPart, 10);
      while (next.length <= idx) next.push({});
      if (i + 2 === parts.length) {
        // Last bracket segment is the array index, next is the leaf key
        break;
      }
      current = next[idx] as Record<string, unknown>;
      i++; // skip the index part
    } else {
      current = next as Record<string, unknown>;
    }
  }

  current[parts[parts.length - 1]] = value;
}
```

// src/cli-app/auth.ts
import { loadTokens, clearTokens, isTokenValid } from '../auth/tokens.js';
import { performOAuth } from '../cli/oauth-flow.js';

export async function handleAuth(command: string): Promise<string> {
  switch (command) {
    case 'login':
      return await authLogin();
    case 'status':
      return await authStatus();
    case 'logout':
      return await authLogout();
    default:
      return `Unknown auth command: ${command}\nUsage: freee auth [login|status|logout]`;
  }
}

async function authLogin(): Promise<string> {
  const result = await performOAuth();
  return `Authentication successful. Access token obtained.`;
}

async function authStatus(): Promise<string> {
  const tokens = await loadTokens();
  if (!tokens) {
    return 'Not authenticated. Run: freee auth login';
  }
  if (!isTokenValid(tokens)) {
    return 'Token expired. Run: freee auth login';
  }
  const expiresAt = new Date(tokens.expires_at).toISOString();
  return `Status: authenticated\nExpires: ${expiresAt}`;
}

async function authLogout(): Promise<string> {
  await clearTokens();
  return 'Logged out. Tokens cleared.';
}
```

// src/cli-app/company.ts
import {
  getCurrentCompanyId,
  setCurrentCompany,
  getCompanyInfo,
} from '../config/companies.js';
import { getValidAccessToken } from '../auth/tokens.js';
import { makeApiRequest } from '../api/client.js';

export async function handleCompany(command: string, args: string[]): Promise<string> {
  switch (command) {
    case 'current':
      return await companyCurrent();
    case 'set':
      return await companySet(args[0]);
    case 'ls':
      return await companyList();
    default:
      return `Unknown company command: ${command}\nUsage: freee company [ls|current|set <id>]`;
  }
}

async function companyCurrent(): Promise<string> {
  const companyId = await getCurrentCompanyId();
  if (!companyId) {
    return 'No company set. Run: freee company set <id>';
  }
  const info = await getCompanyInfo(companyId);
  const name = info?.name ?? 'Unknown';
  return `Current company: ${name} (ID: ${companyId})`;
}

async function companySet(companyId: string | undefined): Promise<string> {
  if (!companyId) {
    return 'Usage: freee company set <company_id>';
  }
  await setCurrentCompany(companyId);
  return `Company set to: ${companyId}`;
}

async function companyList(): Promise<string> {
  const accessToken = await getValidAccessToken();
  if (!accessToken) {
    return 'Not authenticated. Run: freee auth login';
  }

  const response = await makeApiRequest(
    'GET',
    '/api/1/companies',
    {},
    undefined,
    'https://api.freee.co.jp',
  );

  const companies = (response as { companies: Array<{ id: number; name: string; role: string }> }).companies;
  if (!companies || companies.length === 0) {
    return 'No companies found.';
  }

  const lines = companies.map(
    (c) => `  ${c.id}\t${c.name}\t(${c.role})`
  );
  return `Companies:\n${lines.join('\n')}`;
}

// src/cli-app/api-list.ts
import { API_CONFIGS, type ApiType } from '../openapi/schema-loader.js';
import type { MinimalPathItem } from '../openapi/minimal-types.js';

export function listEndpoints(service: ApiType): string {
  const config = API_CONFIGS[service];
  if (!config) {
    return `Unknown service: ${service}`;
  }

  const rows: Array<[string, string, string]> = [];

  for (const [path, pathItem] of Object.entries(config.schema.paths)) {
    const methods = ['get', 'post', 'put', 'delete', 'patch'] as const;
    for (const method of methods) {
      const op = (pathItem as MinimalPathItem)[method];
      if (op) {
        rows.push([method.toUpperCase(), path, op.summary ?? '']);
      }
    }
  }

  rows.sort((a, b) => a[1].localeCompare(b[1]) || a[0].localeCompare(b[0]));

  // Calculate column widths
  const headers: [string, string, string] = ['METHOD', 'PATH', 'SUMMARY'];
  const widths = headers.map((h, i) =>
    Math.max(h.length, ...rows.map(r => r[i].length))
  );

  const formatRow = (row: [string, string, string]) =>
    row.map((cell, i) => cell.padEnd(widths[i])).join('  ');

  return [formatRow(headers), ...rows.map(formatRow)].join('\n');
}

The full OpenAPI schema is needed for docs generation. Load it lazily alongside the minimal schema.

```typescript
// src/cli-app/api-docs.ts
import fs from 'node:fs';
import path from 'node:path';
import { fileURLToPath } from 'node:url';
import { API_CONFIGS, type ApiType } from '../openapi/schema-loader.js';
import type { MinimalPathItem } from '../openapi/minimal-types.js';

const __dirname = path.dirname(fileURLToPath(import.meta.url));

// Map service to full schema filename
const FULL_SCHEMA_FILES: Record<string, string> = {
  accounting: 'accounting-api-schema.json',
  hr: 'hr-api-schema.json',
  invoice: 'invoice-api-schema.json',
  pm: 'pm-api-schema.json',
  sm: 'sm-api-schema.json',
};

// Cache loaded full schemas
const _fullSchemaCache: Record<string, OpenApiSchema> = {};

interface OpenApiSchema {
  paths: Record<string, Record<string, OperationObject>>;
  components?: { schemas?: Record<string, SchemaObject> };
}

interface OperationObject {
  summary?: string;
  description?: string;
  operationId?: string;
  tags?: string[];
  parameters?: ParameterObject[];
  requestBody?: RequestBodyObject;
  responses?: Record<string, ResponseObject>;
}

interface ParameterObject {
  name: string;
  in: string;
  description?: string;
  required?: boolean;
  schema?: SchemaObject;
}

interface RequestBodyObject {
  required?: boolean;
  content?: Record<string, { schema?: SchemaObject }>;
}

interface ResponseObject {
  description?: string;
  content?: Record<string, { schema?: SchemaObject }>;
}

interface SchemaObject {
  type?: string;
  format?: string;
  description?: string;
  required?: string[];
  properties?: Record<string, SchemaObject>;
  items?: SchemaObject;
  enum?: string[];
  example?: unknown;
  minimum?: number;
  maximum?: number;
  $ref?: string;
  oneOf?: SchemaObject[];
  allOf?: SchemaObject[];
  anyOf?: SchemaObject[];
}

// Reuse the same resolution strategy as schema-loader.ts (getSchemasDir)
// to handle all runtime contexts: dist/index.esm.js, bin/freee.js, development (bun)
function getFullSchemasDir(): string {
  const candidates = [
    path.resolve(__dirname, './openapi'),           // dist/ bundled
    path.resolve(__dirname, '../dist/openapi'),     // bin/ bundled
    path.resolve(__dirname, '../../openapi'),       // development (src/cli-app/)
  ];
  for (const c of candidates) {
    // Check for a known full schema file to confirm this is the right dir
    if (fs.existsSync(path.join(c, 'accounting-api-schema.json'))) return c;
  }
  throw new Error(
    `Could not find full OpenAPI schema directory. Searched:\n${candidates.join('\n')}`
  );
}

function loadFullSchema(service: string): OpenApiSchema {
  if (_fullSchemaCache[service]) return _fullSchemaCache[service];
  const dir = getFullSchemasDir();
  const file = FULL_SCHEMA_FILES[service];
  if (!file) throw new Error(`Unknown service: ${service}`);
  const content = fs.readFileSync(path.join(dir, file), 'utf-8');
  _fullSchemaCache[service] = JSON.parse(content);
  return _fullSchemaCache[service];
}

// Resolve actual schema path pattern from a concrete path (e.g., /api/1/deals/123 -> /api/1/deals/{id})
function resolveSchemaPath(service: ApiType, concretePath: string): string | null {
  const config = API_CONFIGS[service];
  if (!config) return null;

  // Exact match
  if (concretePath in config.schema.paths) return concretePath;

  // Pattern match
  for (const schemaPath of Object.keys(config.schema.paths)) {
    const pattern = schemaPath.replace(/\{[^}]+\}/g, '[^/]+');
    if (new RegExp(`^${pattern}$`).test(concretePath)) return schemaPath;
  }
  return null;
}

// ---- Public API ----

export function generateHelp(service: ApiType, concretePath: string): string {
  const schemaPath = resolveSchemaPath(service, concretePath);
  if (!schemaPath) {
    return `Path not found: ${concretePath}\nRun: freee ${service} ls`;
  }

  const config = API_CONFIGS[service];
  const pathItem = config.schema.paths[schemaPath];
  const methods = ['get', 'post', 'put', 'delete', 'patch'] as const;

  const lines: string[] = [`Endpoint: ${schemaPath}`, ''];
  lines.push('Methods:');

  for (const m of methods) {
    const op = (pathItem as MinimalPathItem)[m];
    if (op) {
      lines.push(`  ${m.toUpperCase()}  ${op.summary ?? ''}`);
      lines.push(`    Docs:   freee ${service} ${schemaPath} --docs -X ${m.toUpperCase()}`);
      lines.push(`    Spec:   freee ${service} ${schemaPath} --spec`);
    }
  }

  return lines.join('\n');
}

export function generateDocs(service: ApiType, concretePath: string, method?: string): string {
  const schemaPath = resolveSchemaPath(service, concretePath);
  if (!schemaPath) {
    return `Path not found: ${concretePath}\nRun: freee ${service} ls`;
  }

  const fullSchema = loadFullSchema(service);
  const pathItem = fullSchema.paths[schemaPath];
  if (!pathItem) {
    return `Path not found in full schema: ${schemaPath}`;
  }

  // If method specified, show docs for that method only
  const methods = method
    ? [method.toLowerCase()]
    : Object.keys(pathItem).filter(m => ['get', 'post', 'put', 'delete', 'patch'].includes(m));

  const sections: string[] = [];

  for (const m of methods) {
    const op = pathItem[m] as OperationObject | undefined;
    if (!op) continue;

    sections.push(`### ${m.toUpperCase()} ${schemaPath}`);
    sections.push('');
    if (op.summary) sections.push(`操作: ${op.summary}`);
    if (op.description) {
      // Strip HTML tags for readability
      const desc = op.description.replace(/<[^>]+>/g, '').replace(/\n{3,}/g, '\n\n').trim();
      sections.push('');
      sections.push(`説明: ${desc}`);
    }

    // Parameters
    if (op.parameters && op.parameters.length > 0) {
      sections.push('');
      sections.push('### パラメータ');
      sections.push('');
      sections.push('| 名前 | 位置 | 必須 | 型 | 説明 |');
      sections.push('|------|------|------|-----|------|');
      for (const p of op.parameters) {
        const required = p.required ? 'はい' : 'いいえ';
        const type = formatType(p.schema);
        const desc = formatParamDescription(p);
        sections.push(`| ${p.name} | ${p.in} | ${required} | ${type} | ${desc} |`);
      }
    }

    // Request Body
    if (op.requestBody?.content) {
      const jsonContent = op.requestBody.content['application/json'];
      if (jsonContent?.schema) {
        sections.push('');
        sections.push('### リクエストボディ');
        sections.push('');
        const resolved = resolveRef(jsonContent.schema, fullSchema);
        sections.push(formatSchemaFields(resolved, fullSchema));
      }
    }

    // Response (200 or 201)
    const successCode = op.responses?.['200'] ? '200' : op.responses?.['201'] ? '201' : null;
    if (successCode && op.responses?.[successCode]) {
      const resp = op.responses[successCode];
      const jsonContent = resp.content?.['application/json'];
      if (jsonContent?.schema) {
        sections.push('');
        sections.push(`### レスポンス (${successCode})`);
        sections.push('');
        const resolved = resolveRef(jsonContent.schema, fullSchema);
        sections.push(formatSchemaFields(resolved, fullSchema));
      }
    }

    sections.push('');
  }

  return sections.join('\n');
}

export function generateSpec(service: ApiType, concretePath: string): string {
  const schemaPath = resolveSchemaPath(service, concretePath);
  if (!schemaPath) {
    return JSON.stringify({ error: `Path not found: ${concretePath}` }, null, 2);
  }

  const fullSchema = loadFullSchema(service);
  const pathItem = fullSchema.paths[schemaPath];
  return JSON.stringify(pathItem, null, 2);
}

// ---- Helpers ----

function resolveRef(schema: SchemaObject, root: OpenApiSchema): SchemaObject {
  if (schema.$ref) {
    const refPath = schema.$ref.replace('#/components/schemas/', '');
    const resolved = root.components?.schemas?.[refPath] ?? schema;
    // Recursively resolve in case the target also uses $ref or allOf
    return resolveRef(resolved, root);
  }
  // Handle allOf by merging all schemas into one
  if (schema.allOf) {
    const merged: SchemaObject = { type: 'object', properties: {}, required: [] };
    for (const sub of schema.allOf) {
      const resolved = resolveRef(sub, root);
      if (resolved.properties) {
        merged.properties = { ...merged.properties, ...resolved.properties };
      }
      if (resolved.required) {
        merged.required = [...(merged.required ?? []), ...resolved.required];
      }
    }
    return merged;
  }
  // Handle oneOf/anyOf by picking the first option (best effort)
  if (schema.oneOf?.length) {
    return resolveRef(schema.oneOf[0], root);
  }
  if (schema.anyOf?.length) {
    return resolveRef(schema.anyOf[0], root);
  }
  return schema;
}

function formatType(schema?: SchemaObject): string {
  if (!schema) return 'unknown';
  let t = schema.type ?? 'unknown';
  if (schema.format) t += `(${schema.format})`;
  if (schema.enum) t += ` (選択肢: ${schema.enum.join(', ')})`;
  return t;
}

function formatParamDescription(p: ParameterObject): string {
  let desc = p.description ?? '';
  if (p.schema?.enum) {
    desc += ` (選択肢: ${p.schema.enum.join(', ')})`;
  }
  return desc;
}

function formatSchemaFields(
  schema: SchemaObject,
  root: OpenApiSchema,
  indent = 0,
  maxDepth = 3,
): string {
  if (indent > maxDepth) return '';

  const resolved = resolveRef(schema, root);
  const lines: string[] = [];
  const prefix = '  '.repeat(indent);
  const required = new Set(resolved.required ?? []);

  if (resolved.properties) {
    for (const [name, prop] of Object.entries(resolved.properties)) {
      const resolvedProp = resolveRef(prop, root);
      const req = required.has(name) ? '必須' : '任意';
      const type = formatType(resolvedProp);
      let desc = resolvedProp.description ?? '';
      if (resolvedProp.example !== undefined) desc += ` 例: \`${JSON.stringify(resolvedProp.example)}\``;
      if (resolvedProp.minimum !== undefined) desc += ` (最小: ${resolvedProp.minimum})`;
      if (resolvedProp.maximum !== undefined) desc += ` (最大: ${resolvedProp.maximum})`;

      lines.push(`${prefix}- ${name} (${req}): ${type} - ${desc}`.trimEnd());

      // Recurse into nested objects/arrays
      if (resolvedProp.type === 'object' && resolvedProp.properties) {
        lines.push(formatSchemaFields(resolvedProp, root, indent + 1, maxDepth));
      } else if (resolvedProp.type === 'array' && resolvedProp.items) {
        const resolvedItems = resolveRef(resolvedProp.items, root);
        if (resolvedItems.properties) {
          lines.push(`${prefix}  配列の要素:`);
          lines.push(formatSchemaFields(resolvedItems, root, indent + 2, maxDepth));
        }
      }
    }
  }

  return lines.filter(l => l).join('\n');
}

// src/cli-app/api-exec.ts
import { makeApiRequest } from '../api/client.js';
import { getValidAccessToken } from '../auth/tokens.js';
import { getCurrentCompanyId } from '../config/companies.js';
import { validatePathForService, type ApiType } from '../openapi/schema-loader.js';
import type { ApiInput } from './input-parser.js';

export async function executeApiRequest(
  service: ApiType,
  input: ApiInput,
): Promise<string> {
  // Check auth
  const token = await getValidAccessToken();
  if (!token) {
    return 'Not authenticated. Run: freee auth login';
  }

  // Infer method: explicit > body presence > GET
  const method = input.method ?? (input.body ? 'POST' : 'GET');

  // Validate path
  const validation = validatePathForService(method, input.path, service);
  if (!validation.isValid) {
    return validation.message;
  }

  // Inject company_id into query params if not already set
  const companyId = await getCurrentCompanyId();
  const query: Record<string, string> = { ...input.query };
  if (companyId && !query.company_id) {
    query.company_id = companyId;
  }

  // Inject company_id into body if body exists and doesn't have it
  let body = input.body;
  if (body && companyId && !('company_id' in body)) {
    body = { ...body, company_id: Number(companyId) };
  }

  const result = await makeApiRequest(
    method,
    input.path,
    query,
    body,
    validation.baseUrl,
  );

  // Format output
  if (result === null) {
    return '(no content)';
  }

  if (typeof result === 'object' && 'type' in result && result.type === 'binary') {
    return `File saved: ${(result as { filePath: string }).filePath}`;
  }

  return JSON.stringify(result, null, 2);
}

# freee CLI

## Look things up before answering

The CLI is self-documenting. Always prefer running these commands over guessing
syntax or relying on memorized knowledge:

- `freee accounting ls` -- list every accounting API endpoint.
- `freee accounting <path> --help` -- show methods and usage for an endpoint.
- `freee accounting <path> --docs` -- print formatted API docs (params, response schema).
- `freee accounting <path> --docs -X POST` -- print docs for a specific method.
- `freee accounting <path> --spec` -- print raw OpenAPI fragment (JSON).
- `freee <command> --help` -- help for any command or subcommand.

## Authentication

- `freee auth login` -- OAuth認証（ブラウザが開きます）
- `freee auth status` -- トークン状態確認
- `freee auth logout` -- トークン削除

認証エラーが出たら `freee auth status` で確認し、必要なら `freee auth logout` → `freee auth login` で再認証。

## Company Selection

- `freee company ls` -- 会社一覧
- `freee company current` -- 現在の会社表示
- `freee company set <id>` -- デフォルト会社切り替え

company_id は API リクエストに自動注入されます。明示的に指定した場合はそちらが優先されます。

## API Calls

```bash
# GET with query params
freee accounting /api/1/deals type==income limit==10

# POST with inline body fields
freee accounting /api/1/deals type=expense issue_date=2025-01-15 details[0][tax_code]:=1 details[0][account_item_id]:=101 details[0][amount]:=5000

# POST with JSON body
freee accounting /api/1/deals -d '{"type":"income","issue_date":"2025-01-15","details":[{"tax_code":1,"account_item_id":101,"amount":10000}]}'

# PUT (explicit method)
freee accounting /api/1/deals/123 -X PUT -d '{"type":"expense","details":[...]}'

# DELETE
freee accounting /api/1/deals/123 -X DELETE
```

Input syntax:
- `key==value` -- query parameter
- `key=value` -- body string field
- `key:=json` -- body typed field (numbers, arrays, booleans)
- `-d '{"json":"body"}'` -- full JSON body
- `-X METHOD` -- override HTTP method (GET/POST/PUT/DELETE/PATCH)

## Workflow

1. `freee auth status` で認証状態を確認（未認証なら `freee auth login`）
2. `freee company current` で事業所を確認
3. レシピ（recipes/）で操作手順を確認
4. `freee accounting <path> --docs` でパラメータを確認
5. API を呼び出す

## Error Handling

- 401: トークン期限切れ → `freee auth login` で再認証
- 403: 権限不足またはレートリミット → 数分待って再試行
- 400: パラメータエラー → `--docs` でパラメータを確認

## Recipes

操作手順のガイドは recipes/ ディレクトリを参照:

- `recipes/deal-operations.md` -- 取引（収入・支出）の操作
- `recipes/expense-application-operations.md` -- 経費申請
- `recipes/receipt-operations.md` -- ファイルアップロード
- `recipes/troubleshooting.md` -- トラブルシューティング
